// What it does
⊘

Local-first

Runs entirely on your machine. No API keys to manage, no usage caps, no data transmitted externally. Your prompts stay yours.

/v1

OpenAI-compatible API

Drop-in replacement for OpenAI endpoints. Existing SDKs, scripts, and tools work without modification — just change the base URL.

↔

Model-agnostic

Swap models via config. Point it at any GGUF model file and go. Switch between models without restarting the server.

▶

Built-in playground

Web UI at /ui for testing prompts interactively. No curl required for iteration.
